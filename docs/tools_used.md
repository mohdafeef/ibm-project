## 🕵️ Reconnaissance

| Tool | Description |
|------|-------------|
| [Sublist3r](https://github.com/aboul3la/Sublist3r) | Subdomain enumeration |

| Tool | Purpose |
|------|---------|

| Netcat | Banner grabbing, basic checks |

| Tool | Purpose |
|------|---------|
